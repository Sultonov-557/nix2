# Changelog

All notable changes to Nixul will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Automated ISO builds with Hyprland and Calamares installer
- GitHub Actions workflow for flake checks
- GitHub Actions workflow for automated releases
- Comprehensive installation documentation
- Versioning guide

### Changed
- Repository name from `nix2` to `nixul`
- Enhanced README with ISO installation instructions

## [0.1.0] - Initial Release

### Added
- Initial repository structure
- catppuccin flavor with Catppuccin theme
- macos flavor with macOS-inspired styling
- Modular configuration system
- Home-manager integration
- Stylix theming support
- Developer tools (Docker, PostgreSQL, Redis)
- Basic documentation (README, FAQ, installation guide)

## [0.2.0] - Git and ISO builds

### Added
- Github actions flake checks on push
- ISO installer

## [0.2.1] - Git Optimizations

### Added
- Added 15 level compression to iso build
- Added github actions free space
